分组字段映射（表单 Card → Draft 字段）

CARD 1: BASIC INFO
slug → Input (必填)
title → Input (必填)
subtitle → Input (可选)
shortDescription → Textarea (必填)
CARD 2: TAGS
cloudProviders → TagInput
industries → TagInput
technologies → TagInput
输入逗号分隔，实时渲染 Chip
CARD 3: TYPE & STATUS
assetType → Select (枚举)
status → Select (枚举)
CARD 4: ACCESS CONTROL
visibility → Select (枚举)
allowedRoles → TagInput (从 Roles 拉建议)
allowedUsers → TagInput (从 Users 拉建议)
关键决策：TagInput 组件
分类标签、角色、用户等多值字段统一用 TagInput 组件：
• 输入框支持逗号/空格/回车分隔
• 实时渲染 Chip 预览
• 可选下拉建议（Roles/Users 从 API 拉取）
• 去重去空，输出干净的 string[]
